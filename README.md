Lightweight, nice dev tooling, and blazingly fast. The fastest way to start an app these days.

## Quick Start

1. `package.json`: `name`
2. `start-database.sh`:
   - `DB_CONTAINER_NAME="hello-postgres"`
   - `POSTGRES_DB=hello \`
3. Fill `.env` file
4. `layout.tsx`: change title, description, and icons as you see fit
